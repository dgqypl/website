# 常用系统镜像

展示常用镜像的下载链接。

## 社区镜像

从3.11开始，平台在前端系统镜像控制台加入”社区镜像“的功能，集成了常见的公共可下载的操作系统镜像，用户可以直接点击导入。集成的镜像信息维护在 [这里](https://github.com/yunionio/website/blob/master/static/openimages.yaml)。如果您发现下载链接失效，或者希望增加新的镜像，可以给这个文档提PR。
